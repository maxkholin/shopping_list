# Shopping List App

Приложение для управления списком покупок. 
Пользователи могут добавлять новые элементы, редактировать существующие, отмечать их как выполненные и удалять.
Приложение построено с использованием принципов Чистой Архитектуры.

## Возможности:

- Добавление новых элементов в список покупок.
- Редактирование существующих элементов.
- Удаление и пометка элементов как выполненных.
- Хранение данных в локальной базе данных.

## Технологии:

- **Kotlin** — основной язык разработки приложения.
- **Clean Architecture** — для лучшей организации кода и облегчения тестирования.
- **MVVM** — для обеспечения раздельного управления данными и их отображением.
- **Room** — для работы с локальной базой данных.
- **Data Binding** — для связывания данных с элементами пользовательского интерфейса.


## Структура проекта:

### Data (данные)
- Содержит реализацию базы данных Room (AppDatabase), сущности (ShopItemDbEntity), DAO (ShopListDao) и реализацию репозитория (ShopListRepositoryImpl).
- Включает ShopListMapper для преобразования данных между слоями.

### Domain (домен)
- Содержит бизнес-логику приложения, включая сущность ShopItem и интерфейс ShopListRepository.
- Реализованы юзкейсы для основных операций: AddShopItemUseCase, EditShopItemUseCase, GetShopItemUseCase, GetShopListUseCase, RemoveShopItemUseCase.

### Presentation (представление)
- Отвечает за отображение данных пользователю. Включает MainActivity, ShopItemActivity, фрагменты, адаптеры и ViewModel.
- Реализованы адаптеры для списка покупок и другие вспомогательные классы.
